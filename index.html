How it works

1. Send photos or book a measure-up

Start by sending a few photos, rough dimensions if you have them, and a quick description of what you want.

2. We plan the best setup

We work through layout, function, materials and finish options to suit your space and budget.

3. We build it properly

Your cabinetry is cut on our CNC and built in our Atiamuri workshop using quality materials and precise production.

4. Install or kitset

We can install it for you, or supply a clearly labelled kitset if that suits the job better.
